# tpt-telos-parser

Hand-written lexer, parser, and AST for the [tpt-telos](https://github.com/tpt-solutions/tpt-telos) language.

Parses `.telos` source files into a typed `Vec<Module>` AST that the rest of the tpt-telos pipeline consumes. The grammar is documented in `src/grammar.ebnf`.

## Part of the tpt-telos workspace

See the [main repository](https://github.com/tpt-solutions/tpt-telos) for the full pipeline, examples, and documentation.
